This study has been done as an assignment for the Optimization Methods course.
Two approaches are used: MILP as the exact approach and a Greedy Algorithm as the heuristic approach for portfolio optimization.

Dataset - Historical financial data was collected using the Python library Yahoo Finance through the yfinance API.
In order to download the dataset, first run the download datasets notebook file.

E_MILP contains the implementation of the MILP approach.
The greedy_method file contains the implementation of the Greedy approach.
